### Models (Database Schema)
Located in: `backend/models/`

1. **User.js** - Students, mentors, admins with authentication
2. **Drive.js** - Project drive configurations
3. **Group.js** - Student groups with invitation codes
4. **Synopsis.js** - Project proposals with review workflow
5. **CheckpointSubmission.js** - Progress submissions
